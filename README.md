## 📄 About claude-md-generator

claude-md-generator is a tool that helps you create governance files for your Claude Code projects automatically. It scans your project to find the technology you use, such as programming languages, frameworks, and tools. Then, it builds a clear CLAUDE.md file that explains the project's setup and rules.

You don’t need technical skills or knowledge about coding. The tool works in the background to generate the important documents you need to keep your projects organized and easy to manage.

2. Double-click the icon to open the program.

3. A window will appear, allowing you to select the folder where your Claude Code project is located.

4. Click the "Browse" button and navigate to the folder you want to scan.

5. After selecting the folder, click the "Generate" button.

6. The tool will scan your project files. This may take a few moments.

7. When the process finishes, a new file named `CLAUDE.md` will appear in your project folder.

8. Open the `CLAUDE.md` file with any text editor (like Notepad) to view the generated governance information.

## 🔧 How It Works

The claude-md-generator examines the files inside your project folder. It looks for clues like package files (`package.json`, `requirements.txt`), configuration files, or code patterns.

Using these clues, it understands what your project uses to build and run.

Then, it writes an easy-to-read `CLAUDE.md` file, explaining what technologies your project includes and what rules developers should follow.

This process helps everyone working on the project stay on the same page without extra effort.

No. The installer includes everything needed. Simply download and run the program.

### Can I use claude-md-generator on projects other than Claude Code?

The tool is designed mainly for Claude Code projects. It may work with others but might not detect everything fully.

### Where does claude-md-generator save the CLAUDE.md file?

It saves the file inside the folder you select for scanning.

### Can I update the CLAUDE.md file myself after generation?

Yes. You can open and edit the file with any text editor.

## 🛠️ Troubleshooting

If the generator does not start:

- Check if your computer meets the system requirements.

- Try running the program as an administrator (right-click the icon, then choose "Run as administrator").

If the CLAUDE.md file doesn’t appear:

- Make sure you selected the right project folder.

- Check that the folder is not empty and contains relevant code files.

If the file content looks incomplete:

- Some less common technologies might not be detected automatically.
